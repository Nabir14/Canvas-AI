RenderAI

RenderAI is a scene generation tool which uses Google Gemini AI and Rendering API to generate user requested scenes.


How To Use:

  1. Explain how you want your scene to look like. (Draw natural scenery)
  2. Select your preferred Rendering API
    • HTML Canvas (2d)
    • ThreeJS (3d)
  3. Click Generate Scene to generate your scene.
